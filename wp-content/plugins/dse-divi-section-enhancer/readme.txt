A DIVI Plugin to extends DIVI Sections capabilities. Includes DIVI Youtube Background, DIVI Sidebar, DIVI Scrolbar and more.

== Description ==

### DIVI Enhancer Plugin

DIVI Section Enhancer is a DIVI plugin that works with DIVI Theme or DIVI plugin. Add new options and effects to DIVI Sections. All options are highly customizable and it can be applied easily to any DIVI Section.



### DIVI Section Enhancer Demo Page
[DIVI Section Enhancer Demo](https://pagebuildercode.com/divi-section-enhancer/youtube-background/)





### DIVI Youtube Background
This options lets add any youtube video as background on DIVI sections. DIVI Youtube is applied to section as a fixed element, and it react to scroll movements, so each time user reach a section, background can change, leting you create multiple DIVI sections with youtube videos as background.

[DIVI Youtube Background Demo](https://pagebuildercode.com/divi-section-enhancer/youtube-background/)





### DIVI Sidebar
This options transform any DIVI section in a OFF Canvas Sidebar.It's highly customizable, yo ucan define values as sidebar eight, width, button icons, position, colors, etc....

[DIVI Sidebar Demo](https://pagebuildercode.com/divi-section-enhancer/sidebar/)





### DIVI Scrollbar
This options adjusts the height or width of any DIVI section to add a beautiful scrollbar inside it. It has 6 different scrollbar styles and is highly customizable. Supports vertical or horizontal scrollbars.

[DIVI Scrollbar Demo](https://pagebuildercode.com/divi-section-enhancer/scrollbar)

== Installation ==

Automatic installation is the easiest option as WordPress handles the file transfers itself and you do not need to leave your web browser. To do an automatic install of DIVI Section Enhancer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.



### How to use

After you succesfully installed, the plugin adds new options to sections settings. Open section settings, navigate to advanced tab->custom CSS toggle, scroll down and look for option named "Divi Section Enhancer"




== Frequently Asked Questions ==
### Where can I find the new options?
Open section settings, navigate to advanced tab->custom CSS toggle, scroll down and look for option named "Divi Section Enhancer"

### Why can't I see any change?
Options added by Divi Section Enhancer can't be previewable under visual builder. you must save after changes was applied and check it under front end.
